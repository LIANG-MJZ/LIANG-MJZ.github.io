# [指针](_Study\C_language\1\pointer.md)

- **指针**：只是一个**保存*存储器地址*** 的 **变量**。 

- **指针** 就是 **地址**，**地址** 是 **内存单元** 的 **编号**。

- **指针变量** 就是 **存放地址** 的 **变量** 的 **编号** 

- 指针 **不等于** 指针变量

- **⁎⁎p** = 10; //error  *P为一个整型变量，所以不能在整型变量前再加⁎

- **&** 运算符:可以 **找到** 变量的地址。

- **⁎** 运算符:可以 **读取** 存储器地址中的内容。

- **⁎** 运算符:可以 **设置** 存储器地址中的内容
